# dq9_micro_dwc_server_emulator.cpp

dq9用のexeだけで完結する、dqvcのDLC配布サーバーです。  
配布するコンテンツについては各自で用意してください  

## 注意事項
- インターネットに存在する代替サーバーと同様のDLCを配布する方法は、質問をいただいても答えませんので、頑張って調べてください
- 本ソフトウェアの脆弱性については注意していますが、利用者はMITライセンスに従い、自己責任で本ソフトウェアを使用してください。
  - ウイルスに感染した時に備え、復元ポイントの作成をお勧めします。
- 本ソフトウェアの実行に使用する物理的なWindowsコンピューター、使用するルーターについては、各自の責任をもって用意するものとします。
- 本ソフトウェアは、Windows10以降のパソコン専用です。androidスマホ、iPhoneスマホではサーバーを動かすことはできません。
  - 上級者であれば、clionをインストールし、プログラムを修正し、openssl3.0.*をインストールし、Linuxで動かすことができます。
  - 使用するパソコンがない、パソコンを用意できない、androidスマホで使えないの？などの紛争は言われても困るので連絡してこないでください
